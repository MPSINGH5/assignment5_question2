Question 1: Show the Scope of variables in JS using HTML page

Here are some of the examples of the scope of variable:

  1. var y = 20; function bar() { var y = 200; function baz() { console.log(y); } baz(); } bar(); OUTPUT: 200
  2. function say(message) { if(!message) { let greeting = 'Hello'; console.log(greeting); } console.log(greeting); } say(); OUTPUT:Hello
  3. function getCounter() { 'use strict' counter = 10; return counter; } console.log(getCounter()); OUTPUT: error
